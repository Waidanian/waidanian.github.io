Portfolio and propositions

｢ Using TeX and related software in Web content production ｣

My Organic TeX Demo shows how TeX / LaTeX, Emacs Org-mode and other document formats based on macros can be used to produce high-fidelity content for the Web. The demo, for the initial phase of this ongoing effort, focuses on text content, but I shall show in a soon-to-be-released project that all kinds of graphics can also be produced in the same workflow, bringing to the Web some classic techniques promoting density of information and cognitive efficiency in the tradition of Edward Tufte’s analytic design. The workflow has already been extensively tested in production, in the context of the HU15 group’s publishing work in the digital humanities.

My work with the Lavoie music school (fr) involved teaching the site’s managers how writing directly for the Web in the Markdown format can help them deliver a powerful message for their audience of media-aware students and keen tutors. ｢ Teaching practical recursion to CS students by using industry-relevant, non-numeric data structures ｣

（ Developing the ‘Needledrop Level 0’ Python module ）

While working as a teaching assistant in an introductory programming course designed for Computer Science and Software Engineering students, I realized that students often struggle to grasp the practical importance of recursion when it is first taught to them using binary search trees and suchlike. I designed a new exercise about writing recursive functions that parse narrative XHTML documents, precisely the kind of recursive data structure that is extremely common on the Web. Here is an impression of the resulting module, along with an earnest proposition to write a new module implementing the same algorithms within an industry-standard framework.

The HU15 group is a collective of researchers in the field of French Medieval literature. I designed for them a set of CSS styles that are applied directly to TEI XML manuscript transcriptions (fr). Essayistic pieces

｢ Macros as cognitive catalysts in digital performance ｣

2016-12-21

Consider real-time applications, such as audio signal processing in the context of live performance. One manipulation might be mapped to one operation, such as:

‹Moving one continuous controller› means ‹modulating amplitude proportionally›.

However, if we consider the natural behavior of e.g. acoustic wind instruments, we find ourselves yearning one manipulation mapped to multiple signal processing operations – nature is complex and made of interrelated substances that react together, not in isolation! In this case:

‹If a wind instrument’s sound is louder›, ‹there will probably be more high harmonics› (the sound is brighter).

Amplitude and frequency content change together. So we want this relation:

‹Moving one continuous controller› means that ‹amplitude is modulated proportionally› and ‹harmonics are added according to amplitude increase›.

The manipulation’s quantity acts as parameter for two interrelated operations. The quantity is substituted twice in the resulting operation set. The combined operation, from the performer’s perspective, is one act and can be cognitively apprehended as such, which respects the inherent complexity of the performance and gives the performer more power. The combined operation is a substitution (macro) defined in a statement and becomes a meaningful, useful item in a technical vocabulary. This is an example of ordaining technology to the needs of a given domain, and is the culturally sound thing to do.
